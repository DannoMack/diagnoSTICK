# diagnoSTICK

A bootable USB diagnostic tool that checks the health of a computer's hardware and reports findings in plain English for non-technical users.

## What it checks
- Hard drive and SSD health via SMART data
- CPU temperature
- Battery health (laptops)
- RAM — quick check on boot, full test available if issues are found

## What it doesn't check
- Software problems or Windows issues
- GPU health beyond confirming it's detected
- Anything requiring the machine's operating system to be running

## Requirements
- The target machine must be able to boot from USB
- 64-bit processor
- USB 3.0 recommended for reasonable boot times
